CSS properties

1. background-color

Changes the background color

body { background-color: #f4f4f4; }

2. border-

Adds a border on chosen text

{ border: 1px solid black; }

3. margin-

Used to create space around elements

P { margin-top: 10px; }

4. padding-

used to generate space around content

{ padding-top: 50px; }

5. text-align

Aligns the text to specific direction

{ text-align: center; }

6. font-family

Changes fonts of chosen text

{ font-family: Arial; }

7. display-

specify how an element is shown on pages

{ display: block; }

8. position-

specifies the positioning method of an element

{ position: static; }
